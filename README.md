# Ajay's Cafe - QSR Website

## Short Description
Ajay's Cafe is a professional website for a rapidly expanding Quick Service Restaurant (QSR) chain across India. The website showcases the brand's mission, product offerings, and values while providing an intuitive user experience for customers to learn about the restaurant chain, explore menus, and discover outlet locations.

---

## Overview
Ajay's Cafe is a dynamic QSR chain committed to serving "Good Food For All" - offering delicious and affordable food options including burgers, pizzas, fries, and cold coffee across multiple outlets and cities. This website serves as the digital storefront for the brand.

## Key Features

### 1. **Homepage (index.html)**
- Hero section with brand tagline: "We Serve Food That is Good, Delicious and Accessible to All"
- Embedded YouTube video showcasing the cafe experience
- Featured product sections highlighting pizzas, burgers, and fries
- "Happy Wali Feeling" section emphasizing customer experience
- "Apna Bharosa, Apna Ajay's" trust-building message
- Responsive navbar with navigation links

### 2. **About Us Page (aboutus.html)**
- **Who We Are**: Information about 160+ outlets across 42 cities
- **Founder Information**: Details about Ajay & Jaideep Vasant Solanki
- **Mission Statement**: "To Serve Food That Is Good, Delicious And Accessible To All"
- **Vision Statement**: "To Be The Largest QSR Chain In India by 2030"
- **Core Values**: 6 key principles guiding the business
- **Journey Timeline**: Visual representation of company milestones

### 3. **Menu Page (ourmenu.html)**
- Comprehensive menu showcase with:
  - Burger varieties (Veg, BBQ, Tandoori, Shezwan, Peri-Peri, etc.)
  - Cheese delights and specialty items
  - Detailed descriptions of each dish
  - Visual product images
  - Pricing information

### 4. **Navigation**
- Consistent navbar across all pages with links to:
  - Home
  - About Us
  - Our Menu
  - Our Outlets
  - Franchisee
  - Gallery
  - Careers
  - Blog/News
  - Contact Us
- Mobile-responsive hamburger menu for smaller screens

## Technology Stack

- **Frontend**: HTML5, CSS3
- **Icons**: Font Awesome 6.0.0+ (via CDN)
- **Media**: WebP images for optimized loading, embedded YouTube video
- **Responsiveness**: Mobile-first responsive design with viewport meta tags

---

## Responsive Design
- Hamburger menu for mobile devices (`$BROWSER` compatible)
- Flexible layouts using CSS Grid and Flexbox
- Optimized image formats (WebP) for faster loading
- Meta viewport tag for proper mobile rendering

## Usage Instructions

### Viewing the Website
1. Open `index.html` in a web browser to view the homepage
2. Navigate using the navbar to explore different sections
3. Use the hamburger menu on mobile devices for navigation

### Installing/Deploying
- No special installation required
- Simply clone the repository and open `index.html` in any modern web browser
- Can be deployed to any static web hosting platform (GitHub Pages, Netlify, Vercel, etc.)
